An interactive and beginner-friendly Queue Visualizer built to understand how queues work in Data Structures through smooth animations and real-time operations.

🌟 Features

✨ Interactive Queue Operations
✨ Beautiful Visual Representation
✨ Step-by-Step Animation
✨ Beginner Friendly UI
✨ Fast & Responsive Design
✨ Supports Queue Concepts Clearly

📌 Queue Operations Included
➕ Enqueue

Adds an element to the rear of the queue.

➖ Dequeue

Removes an element from the front of the queue.

👀 Peek / Front

Displays the front element without removing it.

📏 Size

Shows the current number of elements.

🧹 Clear Queue

Removes all elements instantly.

🧠 Concepts Demonstrated
FIFO (First In First Out)
Queue Overflow
Queue Underflow
Front and Rear Pointers
Dynamic Visualization
Real-time Queue Updates
