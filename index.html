🏆 VERDICT
LINEAR QUEUE
✗ Front slots become permanently wasted after each Dequeue.
✗ Rear pointer cannot exceed size — even if front slots are free.
✗ Memory efficiency degrades over time (up to O(n) wasted).
CIRCULAR QUEUE
✓ Rear wraps via modulo arithmetic — reuses freed slots.
✓ Memory is always fully utilized up to actual queue size.
✓ Preferred for buffers, scheduling & real-time systems.
